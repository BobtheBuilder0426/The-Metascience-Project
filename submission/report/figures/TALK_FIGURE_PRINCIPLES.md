# Talk-figure design principles (applied to T1–T5)

Distilled from established guidance on scientific presentation slides (PLOS
Comp Biol "Ten simple rules for effective presentation slides"; MIT Comm Lab;
Alley/McConnell assertion–evidence style). Sources consulted via web search on
2026-07-13; principles are field-standard, not novel.

**One message per slide.** Each talk figure makes exactly one point. If a
manuscript figure has several panels, split it — one panel per slide.

**Title = the takeaway, as a declarative sentence.** Not "Results" or
"exp-002", but the finding itself ("v8 scores highest with every judge").
A viewer who reads only the title should get the point.

**Big, legible type at projection size.** Readable from the back row.
Talk figures here are authored at 16:9 (13.33 × 7.5 in) with a large base
font (≈22 pt title, ≈16–18 pt body) — much larger than the paper figures.

**Minimal text; visuals over words.** Audience should grasp the visual in
~5 seconds. No paragraphs, no dense tables. Short phrases only.

**Strong visual hierarchy + whitespace.** Big element = the message; smaller
supporting text only where needed. Empty space directs the eye.

**≤ 2–3 colours, high contrast, colour-blind-safe.** Same Okabe-Ito
entity→colour map as the paper (black = Baseline, sky-blue = v7, blue = v8,
bluish-green = v9, orange = call-out only). Never red/green opposition.

**Callouts to steer attention.** One arrow / ring / bold annotation for the
key data point — not many.

**Don't reuse paper figures verbatim.** Paper figures are too detailed for a
projector; the talk versions are simplified (fewer marks, larger type, less
axis furniture) while carrying the same true numbers.
